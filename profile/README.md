## 🚀 Getting Started with Packet Analysis

*Prerequisites: A Windows PC with network interface and administrator privileges for packet capture.*

1.  **Install Wireshark:** Complete your wireshark download and run the installer. Install Npcap when prompted for packet capture capabilities.
2.  **Select an Interface:** Launch this packet sniffer and choose the network interface you want to capture traffic from (Wi-Fi or Ethernet).
3.  **Start Capturing:** Click the shark fin icon in this network protocol analyzer to begin capturing live network packets.
4.  **Apply Filters:** Use the display filter bar to isolate specific traffic, such as `http` for web traffic or `ip.addr == 192.168.1.1` for a specific IP.
